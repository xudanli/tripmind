# 🌟 灵感模式测试场景提示词

## 📝 使用说明

这些提示词可用于测试 Inspiration 模式的 AI 生成功能。每个场景都会触发不同的人格类型和体验模板。

---

## 🧪 测试场景

### 1. 🏖️ 海洋光影探索
**提示词**：
```
我想拍海底的光
```

**预期人格**：`photography_exploration` (摄影探索型)  
**预期关键词**：光、镜头、呼吸、流动、视觉  
**预期地点**：马尔代夫、巴厘岛、帕劳  

---

### 2. 🌅 日出冥想之旅
**提示词**：
```
我想一个人静静看日出
```

**预期人格**：`emotional_healing` (情感疗愈型)  
**预期关键词**：慢、安静、陪伴、温柔、平静  
**预期地点**：济州岛、普罗旺斯、冲绳  

---

### 3. 🏔️ 极限攀登挑战
**提示词**：
```

```

**预期人格**：`extreme_exploration` (极限探索型)  
**预期关键词**：挑战、勇气、突破、能量、极限  
**预期地点**：阳朔、泰山、华山  

---

### 4. 👥 文化交流体验
**提示词**：


**预期人格**：`cultural_exchange` (人文交流型)  
**预期关键词**：故事、连接、文化、温度、对话  
**预期地点**：日本、印度、土耳其  

---

### 5. 📸 城市摄影漫步

```
我想记录城市里的光影流动
```

**预期人格**：`urban_creation` (城市创作型)

## 🧬 人格类型对照表

| 人格类型 | 触发关键词 | 语气特征 | 推荐地点 |
|---------|-----------|---------|---------|
| 🎨 urban_creation | 光、构图、建筑、城市 | 低调诗意、视觉化 | 都市、历史街区 |
| 🕊️ emotional_healing | 静、慢、一个人、放松 | 温柔、慢节奏、陪伴感 | 海边、山间、书店 |
| 🔥 extreme_exploration | 挑战、第一次、冲浪、攀岩 | 强节奏、鼓舞、决策导向 | 户外、极限运动地 |
| 💬 cultural_exchange | 认识人、故事、文化、当地 | 温暖对话、叙事感 | 市集、街头、民宿 |
| 📷 photography_exploration | 拍照、光影、记录 | 温柔艺术、对话感 | 自然风光、海岸 |
| 🧠 mind_healing | 放下、心累、疗愈 | 慢节奏、疗愈感 | 森林、寺庙、自然 |
| 🌿 nature_discovery | 自然、探索、发现 | 轻盈自然、流动感 | 国家公园、自然保护区 |
