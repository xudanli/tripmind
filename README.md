# 🌍 AI 旅行伴侣

一个基于 Vue 3 + TypeScript + Vite 的智能旅行规划应用，提供两种不同的旅行规划模式。

## ✨ 功能特点

### 🚀 Planner 模式 - 理性出发
- **目标明确**：适合已有明确目的地的用户
- **高效规划**：基于目的地、时间、预算、偏好生成详细行程
- **专业建议**：提供住宿、交通、美食等专业推荐

### 🌸 Seeker 模式 - 随心而行  
- **情感导向**：基于用户当前心情和情感需求
- **治愈推荐**：为疲惫、想逃离、需要治愈的用户找到合适目的地
- **温柔陪伴**：提供情感化的旅行建议和陪伴

## 🛠️ 技术栈

- **前端框架**：Vue 3 + TypeScript
- **构建工具**：Vite
- **UI 组件库**：Ant Design Vue
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **样式**：CSS3 + 渐变背景

## 🚀 快速开始

### 环境要求
- Node.js 20.19+ 或 22.12+
- npm 或 yarn

### 安装依赖
```bash
npm install
```

### 启动开发服务器
```bash
npm run dev
```

### 构建生产版本
```bash
npm run build
```

### 预览生产版本
```bash
npm run preview
```

## 📱 页面结构

- **首页** (`/`) - 模式选择页面
- **Planner 模式** (`/planner`) - 理性规划流程
- **Seeker 模式** (`/seeker`) - 情感导向流程  
- **结果页面** (`/results`) - 显示生成的行程规划

## 🎨 设计特色

- **渐变背景**：Planner 模式使用蓝紫色渐变，Seeker 模式使用粉红色渐变
- **响应式设计**：完美适配桌面端和移动端
- **动画效果**：流畅的页面切换和交互动画
- **情感化设计**：Seeker 模式注重情感表达和治愈感

## 🔧 开发说明

### 状态管理
使用 Pinia 管理全局状态，包括：
- 用户表单数据
- AI 生成的行程数据
- 加载状态和错误处理

### 组件结构
```
src/
├── components/          # 公共组件
├── views/              # 页面组件
├── stores/             # 状态管理
├── router/             # 路由配置
└── assets/             # 静态资源
```
