# 五段式心智流·深化版叙事蓝图 实施计划

## 📋 概述

按照"五段式心智流"重构灵感模式详情页，将用户旅程从"信息浏览"转变为"心理体验"。

---

## 🎯 五段式架构

### ① 召唤——让好奇发声
**状态**: ✅ 已完成
**位置**: `InspirationHero.vue`
**实现**:
- 全屏视频背景（已实现动态背景图）
- 诗意短句："你是否也在等待，一场与你自己的相遇？"
- 滚动提示："轻轻进入 ↓"
- 呼吸式能量圈动画

### ② 映照——被看见的瞬间
**状态**: 🔄 待实现
**位置**: 在 `ExperienceDay.vue` 中，替换现有的 Mood Space
**实现要点**:
- 心境共鸣卡片滑动区
- 每张卡片是一句情绪识别语
- 背景声轻淡（心跳/呼吸音）
- 用户选择"触动语录"后自动过渡到下阶段

**数据来源**:
- 从 `travel.value?.data?.detectedIntent?.emotionTone` 生成情绪识别语
- 从 `highlights` 中提取情感关键词

### ③ 对话——觉醒的契机
**状态**: 🔄 待实现
**位置**: 替换现有的 Journey Flow
**实现要点**:
- 四大支柱模块化呈现：
  - 脱离（Departure）
  - 情境（Context）
  - 内化（Internalization）
  - 转化（Transformation）
- 每个模块以"如果…"或"当…"句式的问题引导
- 点击"思考提问"弹出短视频或文字触发深层反思

### ④ 内化——安静的沉淀
**状态**: 🔄 待实现
**位置**: 替换现有的 Creative Zone
**实现要点**:
- 页面节奏减缓（更慢的滚动、更宽松的间距）
- "写信给未来自己"交互框（匿名保存）
- 轻微背景声 + 日出/波浪影像
- 用户在留白中自我对话

### ⑤ 转化——旅程的延续
**状态**: 🔄 待实现
**位置**: 重构 Echo Outro
**实现要点**:
- "真实旅人故事卡"展示
- "挑战计划"模块
- "加入同路人"社群入口
- 尾声动画："旅程，从现在开始。"

## 📝 数据要求

需要在 AI 生成灵感数据时，生成以下字段：

```typescript
interface InspirationData {
  // ... 现有字段
  
  // ② 映照阶段
  emotionReflections?: Array<{
    text: string        // 情绪识别语
    emotion: string     // 情绪类型
    intensity: number   // 强度 0-1
  }>
  
  // ③ 对话阶段
  fourPillars?: {
    departure: {
      question: string
      media?: string    // 视频/图片URL
      reflection: string
    }
    context: {
      question: string
      media?: string
      reflection: string
    }
    internalization: {
      question: string
      media?: string
      reflection: string
    }
    transformation: {
      question: string
      media?: string
      reflection: string
    }
  }
  
  // ④ 内化阶段
  innerDialoguePrompts?: string[]  // 内化提示语
  
  // ⑤ 转化阶段
  travelerStories?: Array<{
    name: string
    avatar: string
    story: string
    location: string
  }>
  challengePlan?: {
    title: string
    steps: string[]
    deadline?: string
  }
}
```

## 🎨 设计原则

1. **渐进式披露**: 内容逐步显现，避免一次性展示
2. **情绪共鸣**: 每个阶段都要触动用户情绪
3. **自然过渡**: 阶段之间平滑过渡，无明显割裂
4. **沉浸式体验**: 全屏、背景声、动效配合
5. **用户控制**: 用户可以选择跳过或深入某个阶段
